# FitForge

A comprehensive workout tracker with exercise library, custom routine builder, active workout mode with rest timer, and progress analytics

Built with [Builddy](https://builddy.dev) — AI-powered app builder using GLM 5.1.

## Features

- exercise-library-with-muscle-group-filter
- pre-built-workout-templates
- custom-workout-builder-with-drag-drop
- active-workout-mode-with-rest-timer
- haptic-visual-feedback-on-set-completion
- progress-tracking-with-charts
- workout-calendar-with-streaks
- personal-records-dashboard
- dark-mode-for-gym-environments
- user-authentication-and-data-persistence

## Quick Start

### Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

### Docker

```bash
docker compose up
```

### Deploy to Railway/Render

1. Push this directory to a GitHub repo
2. Connect to Railway or Render
3. It auto-detects the Dockerfile
4. Done!

## Tech Stack

- **Frontend**: HTML/CSS/JS + Tailwind CSS
- **Backend**: Express.js
- **Database**: SQLite
- **Deployment**: Docker
